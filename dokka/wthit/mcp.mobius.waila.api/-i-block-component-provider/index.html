@ApiSide.ClientOnly()

public interface IBlockComponentProvider(source)

Used to provide Block/BlockEntity tooltip information to Waila.

All methods in this interface shouldn't to be called by the implementing mod. An instance of the class is to be registered via the IRegistrar instance provided in IWailaPlugin.

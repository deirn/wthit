public void pick(IPickerAccessor accessor, IPickerResults results, IPluginConfig config)(source)

Returns the object that Waila will show the tooltip to, typically by raycasting from the camera's eye.

Note:return all objects in a line from cast origin up until max distance, that way Waila can try to show tooltip for further object if the tooltip for nearer object is disabled for some reason.

Check for config value to save some processing time.
